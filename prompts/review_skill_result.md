# Skill 执行复盘提示词模板

> 用途：手动对一次 skill 执行进行复盘时使用的提示词。

---

## 模板

```
请对本次 skill 执行进行复盘，读取 runs/ 中最新的执行记录，输出以下内容：

## 复盘要求
1. 分析成功点和错误点
2. 评估 3 个维度（各 0-5 分）：任务完成度、代码正确性、解释清晰度
3. 判断是否需要人工返工
4. 判断是否需要更新 knowledge/
5. 判断是否需要优化对应 skill（通过 harmony-skill-evolver 判断）

## 要求
- 不要因为单次偶然失败就建议大改 skill
- 先排除外部因素（需求不清、环境问题等）
- 优化建议要具体，说明改哪个文件的什么内容
```

---

## 使用场景

- 做完一个功能后想回顾一下
- 感觉某个 skill 不太好用，想分析问题
- 想看看最近几次执行的得分趋势
