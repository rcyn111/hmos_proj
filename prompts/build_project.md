# 项目构建提示词模板

> 用途：当用户要从零开始构建一个 HarmonyOS App 时使用的提示词。

---

## 模板

```
你是一个 HarmonyOS 开发专家。请根据以下需求，帮我规划一个完整的鸿蒙应用开发方案。

## 我的需求
{用户需求描述}

## 我的背景
- HarmonyOS 开发新手
- 使用 DevEco Studio
- 目标 HarmonyOS 版本：{版本}
- 目标设备：{手机 / 平板 / 车机}

## 请输出
1. 需求理解（你的理解是否正确）
2. PRD 摘要
3. 功能列表（拆成小功能）
4. 页面结构和导航关系
5. 技术方案（用哪些 ArkUI 组件 / 存储方式 / 权限）
6. 开发阶段（分 3-5 个阶段，每阶段 2-4 个任务）
7. 当前最小任务（现在就要做的第一步）

## 要求
- 用中文回答
- 每次只做一个小功能
- 涉及的新概念要解释
- 不确定的 API 要标记，不要编造
- 代码修改前先说明文件路径和原因
```

---

## 使用方式

1. 将 `{用户需求描述}` 替换为实际需求
2. 将 `{版本}` 替换为目标 HarmonyOS 版本（如 5.0）
3. 将 `{设备}` 替换为目标设备类型
4. 粘贴到 Claude Code 对话中
