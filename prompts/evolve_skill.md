# Skill 进化提示词模板

> 用途：手动触发 skill 进化分析时使用的提示词。

---

## 模板

```
请使用 harmony-skill-evolver 分析以下 skill 是否需要优化。

## 目标 Skill
{skill-name}

## 参考执行记录
- runs/{记录 1}
- runs/{记录 2}
- runs/{记录 3}

## 分析重点
1. 是否有规律性的失败模式？
2. 执行步骤的顺序是否合理？
3. 输出格式是否满足需求？
4. 成功标准/失败处理是否完善？
5. 哪些问题来自 skill 本身（而非外部因素）？

## 要求
- 至少 2 次同类问题才建议优化
- 优化建议要给出修改前后的对比
- 记录到 memory/skill_evolution_log.md
```

---

## 使用场景

- 积累了几次执行记录后，手动触发进化分析
- 用户明确觉得某个 skill 不好用，想定位问题
- 定期检查 skill 质量
